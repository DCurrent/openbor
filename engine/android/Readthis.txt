OpenBOR for android, ported by UTunnels (utunnels@hotmail.com).

Precompiled libs include sdl 2.0, zlib 1.2.3, libogg 1.3.0, libpng 1.5.6 and libvorbis 1.3.2

Source code for those libs can be found on internet.

To compile this you need to install android sdk and ndk properly, read more at http://developer.android.com/index.html

Make sure you change the sdk path in local.properties according to your own situation.
