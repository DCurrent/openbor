listmusic

Description
--------------------------------------------------------------------------------
Listmusic is a simple command line utility to list all of the BOR format music 
files in a PAK file so you can play them out of the PAK file with borplay.

Instructions
--------------------------------------------------------------------------------
Windows: Use the listmusic.exe file in the windows directory.

Linux: Extract the tarfile in the linux directory and use the script "listmusic".
       You must have Python installed.

Mac: Extract the tarfile in the mac directory and use the script "listmusic".
